# Projeto EDA FC - Simulador de Gestão de Futebol

Este projeto consiste no desenvolvimento de um sistema em **C++** para simular o funcionamento do "EDA FC", focando-se na gestão de plantéis e dinâmica de jogo.

## 🎯 Objetivos do Projeto
O sistema gere múltiplas listas de jogadores e aplica regras de restrição, garantindo a reorganização eficiente de **arrays dinâmicos**.

### Funcionalidades Principais:
- **Gestão de Plantéis:** Controlo de 11 Titulares e 6 Suplentes.
- **Dinâmica de Jogo:** Simulação de Jornadas, Suspensões e Lesões.
- **Mercado:** Sistema de Transferências entre listas.

## Restrições Técnicas (Obrigatório)
Para este projeto, a implementação deve focar-se em estruturas de dados manuais:
- [x] Utilização de **structs** e **arrays dinâmicos** definidos pelos alunos.
- [ ] ❌ Proibido: Classe `std::vector`.
- [ ] ❌ Proibido: Listas ligadas.
- [ ] ❌ Proibido: Bibliotecas `Stack` ou `Queue`.
